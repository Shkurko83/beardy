Cómo usar Beardy

Esta guía está escrita en lenguaje sencillo. Si nunca ha usado Markdown, no pasa nada. Siga los pasos en orden.

¿Qué es Beardy y qué es Markdown?

Beardy es una aplicación para Mac en la que puede escribir y leer cómodamente archivos de texto en formato Markdown.

Markdown no es un programa aparte: es una forma de marcar el texto. Por ejemplo:

  • # Título — título grande
  • **negrita** — texto en negrita
  • - elemento — punto de lista

Beardy muestra las «marcas» (símbolos) o el aspecto final — como en una web o en un PDF.

Primer uso — qué pulsar

Al abrir Beardy verá una pantalla de bienvenida con tres acciones:

  1. Documento nuevo — página en blanco para texto nuevo.
  2. Abrir archivo — elegir un archivo que ya tenga en el Mac (a menudo con extensión .md).
  3. Importar — arrastrar un archivo desde el Finder a la ventana de Beardy.

Archivos recientes: debajo de los botones puede haber una lista de lo que abrió antes. Pulse el nombre — el archivo se abrirá de nuevo. El botón Borrar solo vacía la lista; los archivos en el disco no se eliminan.

Cómo escribir y dar formato al texto

En la parte superior hay una barra con botones. Insertan el formato por usted:

  • negrita y cursiva;
  • títulos y listas;
  • enlace, imagen, bloque de código.

También puede escribir los símbolos usted mismo — por ejemplo **palabra** para negrita.

Puede deshacer un error: menú «Edición» → «Deshacer» o el atajo de Mac (véase más abajo).

Cuatro modos — qué significan

Arriba puede cambiar de modo. Es el mismo texto — solo cambia cómo lo ve.

Edit

Se ve el texto normal con símbolos (#, **, etc.). Útil cuando hay mucho código o corrige el formato a mano.

Live

El texto se ve «bonito» al momento — títulos grandes, negrita de verdad. Cómodo para artículos y notas.

Split

A la izquierda el texto con símbolos, a la derecha la vista previa. Bueno para aprender y documentos largos.

Preview

Solo la vista final, sin símbolos. Útil antes de imprimir o exportar.

Consejo: en documentos largos use el modo Split y en «Ajustes → Apariencia» active «desplazamiento sincronizado» para que la parte izquierda y la derecha se muevan juntas.

Cómo guardar

Beardy guarda su trabajo como un archivo de texto normal en el Mac.

  1. Menú Archivo → Guardar (o el atajo habitual de Mac — véase abajo).
  2. En Ajustes → General puede activar guardado automático — el programa guarda solo cada 10–120 segundos.

Si inserta imágenes, la app puede copiarlas a la carpeta del documento — así el archivo se abrirá bien en otro Mac también (ajuste en «Ajustes → Editor»).

Lista de títulos (Outline)

Si el texto tiene títulos (#, ##, …), abra en la barra el botón de estructura / títulos.

Aparecerá una lista de secciones. Pulse un nombre — el documento se desplazará hasta ese sitio. Puede arrastrar el borde del panel para hacerlo más ancho o estrecho.

Varios documentos a la vez

Puede abrir varios archivos en pestañas (como en el navegador). Cada pestaña tiene su propio texto. Al cambiar de pestaña no se borra el historial de «deshacer» de cada archivo.

Colores y tema

¿Quiere pantalla oscura o clara? Abra la lista de temas en el menú o la barra (en la app puede llamarse «Themes»). Elija lo que sea más cómodo para la vista.

En este sitio, el botón ◐ arriba solo cambia el tema claro/oscuro de la web — no el de Beardy.

Ajustes — dónde está cada cosa

Abra Ajustes en el menú de Beardy. Las secciones principales:

  • General — guardado automático: activado/desactivado y cada cuánto.
  • Editor — tamaño de fuente, números de línea, tabulaciones o espacios.
  • Entrada — corrección ortográfica; «comillas inteligentes» (para textos técnicos suele ir mejor desactivadas).
  • Apariencia — en modo Split, desplazar texto y vista previa juntos.
  • Exportar — opciones de PDF e imágenes.

Atajos de teclado (en Mac)

La tecla ⌘ es Comando (símbolo ⌘, junto a la barra espaciadora). En Beardy hay combinaciones habituales:

  • ⌘N — documento nuevo
  • ⌘O — abrir archivo
  • ⌘S — guardar
  • ⌘Z — deshacer · ⌘⇧Z — rehacer
  • ⌘B negrita · ⌘I cursiva
  • ⌘K — lista completa de atajos dentro de la app

No hace falta memorizarlos — todo se puede hacer con el ratón y los menús.

Exportar a PDF y otros formatos

Cuando el texto esté listo:

  1. Abra el menú Archivo (o «Compartir»).
  2. Elija el formato: PDF, página web (HTML), imagen (PNG) o Word (DOCX).

Para un archivo de libro (EPUB) hace falta el programa gratuito Pandoc en el Mac.

Fórmulas y diagramas (si los necesita)

Es opcional. Para una fórmula en el texto se usan signos de dólar, por ejemplo $E = mc^2$. Para un diagrama sencillo — un bloque de código con la palabra mermaid. La app los muestra en la vista previa sin Internet.

Si no necesita fórmulas ni diagramas — puede saltarse esta sección.

Para Mac · Notas y documentos

Un editor claro para archivos Markdown

Markdown es una forma sencilla de escribir: títulos, listas y enlaces con caracteres normales (por ejemplo # para un título). Beardy muestra cómo se verá el texto — al escribir o en otra ventana.

Descargar Beardy Cómo usar la aplicación

Tus archivos quedan en tu Mac. La app puede guardar sola. La vista previa funciona sin Internet.

Empezar en tres pasos

No hace falta curso — abre la app y escribe.

  1. Instala Beardy y ábrelo. Verás: documento nuevo, abrir archivo o arrastrar un archivo a la ventana.
  2. Escribe. Barra de herramientas para negrita, título, lista — o escribe los símbolos Markdown.
  3. Elige la vista: solo símbolos, solo formato, o ambos a la vez. Luego guarda o exporta a PDF.

Guía paso a paso completa

¿Para quién es Beardy?

Para quien escribe textos largos, instrucciones o documentación en Mac.

Programadores

  • El código en el texto se colorea
  • Lista de títulos — clic para ir a la sección
  • Deshacer funciona al cambiar de vista
  • Varios archivos en pestañas; autoguardado

Autores de artículos y guías

  • Como un editor normal — el formato se ve al instante
  • Guardar en PDF, página web, imagen o Word
  • Fórmulas y diagramas simples (vista previa sin red)
  • Corrección ortográfica

Qué puede hacer Beardy

Lo principal — en palabras sencillas.

◇

Cuatro vistas

Solo texto, solo formato, ambos, o solo resultado. Deshacer se mantiene.

☰

Botones de la barra

Negrita, cursiva, títulos, listas, enlaces, código — un clic.

◫

Índice de títulos

Lista de secciones — clic para ir allí.

∑

Fórmulas y diagramas

Mates o diagrama en vista previa (sin red).

◐

Claro y oscuro

Tema claro, oscuro o como el Mac.

↗

Exportar

PDF, web, imagen o Word. EPUB con Pandoc gratis.

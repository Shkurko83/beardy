Как пользоваться Beardy

Инструкция написана простым языком. Если вы никогда не работали с Markdown — ничего страшного. Делайте шаги по порядку.

Что такое Beardy и что такое Markdown?

Beardy — программа для Mac, в которой удобно писать и читать текстовые файлы в формате Markdown.

Markdown — это не отдельная программа, а способ пометить текст. Например:

  • # Заголовок — большой заголовок
  • **жирный** — жирный текст
  • - пункт — пункт списка

Beardy показывает либо эти «метки» (символы), либо готовый вид — как на сайте или в PDF.

Первый запуск — что нажать

После открытия Beardy вы видите экран приветствия с тремя действиями:

  1. Новый документ — пустая страница для нового текста.
  2. Открыть файл — выбрать уже существующий файл на Mac (часто с окончанием .md).
  3. Импорт — перетащить файл из Finder в окно Beardy.

Недавние файлы: ниже кнопок может быть список того, что вы открывали раньше. Нажмите на название — файл откроется снова. Кнопка Очистить убирает только список; сами файлы на диске не удаляются.

Как писать и оформлять текст

Вверху окна есть панель с кнопками. Они вставляют оформление за вас:

  • жирный и курсив;
  • заголовки и списки;
  • ссылку, картинку, блок кода.

Можно и самому вводить символы — например **слово** для жирного.

Ошибку можно отменить: меню «Правка» → «Отменить» или сочетание клавиш Mac (см. раздел ниже).

Четыре режима — что они значат

Вверху можно переключить режим. Это один и тот же текст — меняется только то, как вы его видите.

Edit

Виден обычный текст с символами (#, ** и т. д.). Удобно, когда много кода или вы правите разметку вручную.

Live

Текст сразу выглядит «красиво» — заголовки крупные, жирный жирный. Удобно для статей и заметок.

Split

Слева — текст с символами, справа — как он будет выглядеть. Хорошо для обучения и длинных документов.

Preview

Только готовый вид, без символов. Удобно перед печатью или экспортом.

Совет: для длинного документа включите режим Split и в «Настройки → Оформление» — «синхронная прокрутка», чтобы левая и правая части двигались вместе.

Как сохранить

Beardy сохраняет работу обычным текстовым файлом на вашем Mac.

  1. Меню Файл → Сохранить (или стандартное сочетание Mac — см. ниже).
  2. В Настройки → Основные можно включить автосохранение — тогда программа сама сохраняет каждые 10–120 секунд.

Если вставляете картинки, приложение может копировать их в папку документа — тогда файл откроется правильно и на другом Mac (настройка в «Настройки → Редактор»).

Список заголовков (Outline)

Если в тексте есть заголовки (#, ##, …), откройте на панели кнопку структуры / заголовков.

Появится список разделов. Нажмите на название — документ прокрутится к этому месту. Край панели можно потянуть, чтобы сделать её шире или уже.

Несколько документов сразу

Можно открыть несколько файлов во вкладках (как в браузере). В каждой вкладке свой текст. Переключение вкладок не стирает историю «отменить» в каждом файле.

Цвета и тема

Хотите тёмный или светлый экран? Откройте список тем в меню или на панели (в программе это может называться «Themes»). Выберите то, что приятнее для глаз.

На этом сайте кнопка ◐ вверху переключает светлую/тёмную тему только для сайта — не внутри Beardy.

Настройки — что где лежит

Откройте Настройки в меню Beardy. Основные разделы:

  • Основные — автосохранение: вкл/выкл и как часто.
  • Редактор — размер шрифта, номера строк, табы или пробелы.
  • Ввод — проверка орфографии; «умные кавычки» (для технических текстов часто лучше выключить).
  • Оформление — в режиме Split прокручивать текст и просмотр вместе.
  • Экспорт — параметры PDF и картинок.

Горячие клавиши (на Mac)

Клавиша ⌘ — это Command (значок ⌘, рядом с пробелом). В Beardy есть привычные сочетания:

  • ⌘N — новый документ
  • ⌘O — открыть файл
  • ⌘S — сохранить
  • ⌘Z — отменить · ⌘⇧Z — вернуть
  • ⌘B жирный · ⌘I курсив
  • ⌘K — полный список сочетаний внутри программы

Запоминать их не обязательно — всё можно сделать мышкой и через меню.

Экспорт в PDF и другие форматы

Когда текст готов:

  1. Откройте меню Файл (или «Поделиться»).
  2. Выберите формат: PDF, веб-страница (HTML), картинка (PNG) или Word (DOCX).

Для файла книги (EPUB) нужна бесплатная программа Pandoc на Mac.

Формулы и схемы (если понадобятся)

Это необязательно. Для формулы в тексте используют знаки доллара, например $E = mc^2$. Для простой блок-схемы — блок кода со словом mermaid. Программа покажет их в просмотре без интернета.

Если формулы и схемы не нужны — этот раздел можно пропустить.

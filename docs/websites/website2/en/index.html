For Mac · For notes and documents

A clear editor for Markdown files

Markdown is a simple way to write text: headings, lists, and links are marked with ordinary characters (for example # for a heading). Beardy shows how your text will look — while you type or in a separate window.

Download Beardy How to use the app

Your files stay on your Mac. The app can save changes automatically. Preview works without the internet.

How to start in three steps

No manual or special training needed — open the app and write.

  1. Install Beardy on your Mac and open it. You will see buttons: create a new document, open a file, or drag a file into the window.
  2. Write your text. Use the toolbar for bold, headings, and lists — or type Markdown symbols yourself.
  3. Choose how to view: text only, formatted text only, or text and preview side by side. Save or export to PDF when ready.

Who is Beardy for?

For anyone who writes long texts, instructions, or code documentation on a Mac.

Programmers

  • Code in documents is highlighted in color
  • A list of headings on the side — click to jump to a section
  • Undo works even if you switch the view mode
  • Several files open in tabs; auto-save available

Authors of articles and guides

  • Edit like in a normal editor — formatting appears right away
  • Save as PDF, web page, image, or Word file
  • Formulas and simple charts in the text (no internet needed for preview)
  • Spell check while typing

What Beardy can do

The main features — in plain language.

◇

Four view modes

Text only, formatted text, text + preview, or preview only. Switch anytime — your undo history is kept.

☰

Buttons on the toolbar

Bold, italic, headings, lists, links, and code — insert with one click if you do not want to remember symbols.

◫

Table of contents

A panel lists your headings. Click a line — the document scrolls to that place.

∑

Formulas and charts

If you need math or a flowchart in the text — the app can show them in preview (works offline).

◐

Light and dark appearance

Choose a comfortable theme for your eyes — light, dark, or follow the Mac system setting.

↗

Export

Save the result as PDF, web page, picture, or Word. Books (EPUB) — if you install the free Pandoc program.

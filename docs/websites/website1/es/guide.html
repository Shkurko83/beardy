Cómo usar BlackBeard Editor

Esta guía está escrita en lenguaje sencillo. Si nunca has usado Markdown, no pasa nada. Sigue los pasos en orden, capitán.

¿Qué es BlackBeard Editor y qué es Markdown?

BlackBeard Editor es un programa para Mac que ayuda a escribir y leer archivos de texto en formato Markdown.

Markdown no es un programa aparte: es una forma de marcar el texto. Por ejemplo:

  • # Título — encabezado grande
  • **negrita** — texto en negrita
  • - elemento — viñeta de lista

BlackBeard Editor muestra las «marcas» (símbolos) o el aspecto final — como en la web o en un PDF.

Primer arranque — qué pulsar

Al abrir BlackBeard Editor verás la pantalla de bienvenida con tres acciones:

  1. New Document — página en blanco para texto nuevo.
  2. Open File — abrir un archivo existente en tu Mac (a menudo con extensión .md).
  3. Import — arrastrar un archivo desde Finder a la ventana de BlackBeard Editor.

Archivos recientes: debajo de los botones puede aparecer una lista de lo que abriste antes. Haz clic en un nombre para abrirlo de nuevo. El botón Clear solo borra la lista; los archivos en disco no se eliminan.

Cómo escribir y dar formato

Arriba en la ventana hay una barra de herramientas con iconos que insertan el formato por ti:

  • negrita y cursiva;
  • títulos y listas;
  • enlace, imagen, bloque de código.

También puedes escribir los símbolos tú mismo — por ejemplo **palabra** para negrita.

Para deshacer un error: menú Edición → Deshacer, o el atajo habitual de Mac (ver más abajo).

Cuatro modos — qué significa cada uno

Arriba puedes cambiar de modo. Es el mismo documento; solo cambia cómo lo ves.

Edit

Ves texto plano con símbolos (#, **, etc.). Ideal cuando hay mucho código o editas la sintaxis a mano.

Live

El texto se ve «acabado» mientras escribes: títulos grandes, negrita de verdad. Cómodo para artículos y notas.

Split

A la izquierda el texto con símbolos, a la derecha la vista previa. Bueno para aprender Markdown y documentos largos.

Preview

Solo el aspecto final, sin símbolos. Cómodo antes de imprimir o exportar.

Consejo: en documentos largos usa Split y activa el desplazamiento sincronizado en Settings → Appearance para que ambos lados se muevan juntos.

Cómo guardar

BlackBeard Editor guarda tu trabajo como archivo de texto normal en tu Mac.

  1. Menú File → Save (o el atajo estándar de Mac — ver abajo).
  2. En Settings → General puedes activar el guardado automático: la app guarda sola cada 10–120 segundos.

Si insertas imágenes, la app puede copiarlas a la carpeta del documento; así el archivo se abre bien en otro Mac también (ajuste en Settings → Editor).

Lista de títulos (Outline)

Si tu documento tiene títulos (#, ##, …), abre el panel Outline en la barra de herramientas.

Verás una lista de secciones. Haz clic en un nombre y el documento se desplazará hasta ahí. Puedes arrastrar el borde del panel para hacerlo más ancho o estrecho.

Varios documentos a la vez

Puedes abrir varios archivos en pestañas (como en un navegador). Cada pestaña es un documento distinto. Cambiar de pestaña no borra el historial de deshacer de cada archivo.

Colores y tema

¿Pantalla oscura o clara? Abre la lista Themes en el menú o la barra de herramientas. Elige lo que descanse la vista.

En este sitio, el botón ◐ de arriba solo cambia el tema de la web, no el de BlackBeard Editor.

Settings — dónde está cada cosa

Abre Settings desde el menú de BlackBeard Editor. Pestañas principales:

  • General — guardado automático: activado/desactivado y cada cuánto.
  • Editor — tamaño de fuente, números de línea, tabulaciones o espacios.
  • Typing — corrección ortográfica; «comillas inteligentes» (a menudo mejor desactivadas en texto técnico).
  • Appearance — desplazar texto y vista previa juntos en modo Split.
  • Export — opciones para PDF e imágenes.

Atajos de teclado (en Mac)

La tecla ⌘ se llama Command (símbolo ⌘, junto a la barra espaciadora). BlackBeard Editor usa combinaciones habituales:

  • ⌘N — New Document
  • ⌘O — Open File
  • ⌘S — guardar
  • ⌘Z — deshacer · ⌘⇧Z — rehacer
  • ⌘B negrita · ⌘I cursiva
  • ⌘K — lista completa de atajos dentro de la app

No hace falta memorizarlos: todo se puede hacer con el ratón y los menús.

Exportar a PDF y otros formatos

Cuando el texto esté listo:

  1. Abre el menú File (o Compartir).
  2. Elige el formato: PDF, página web (HTML), imagen (PNG) o Word (DOCX).

Para un archivo de libro (EPUB) necesitas el programa gratuito Pandoc instalado en tu Mac.

Fórmulas y diagramas (si los necesitas)

Es opcional. Para una fórmula en el texto usa signos de dólar, por ejemplo $E = mc^2$. Para un diagrama sencillo — un bloque de código con la palabra mermaid. La app los muestra en Preview sin internet.

Si no necesitas mates ni diagramas, salta esta sección: el barco navega igual.

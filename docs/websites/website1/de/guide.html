Wie man BlackBeard Editor nutzt

Diese Anleitung ist in einfacher Sprache geschrieben. Wenn Sie noch nie mit Markdown gearbeitet haben — kein Problem. Gehen Sie die Schritte der Reihe nach durch, Kapitän.

Was ist BlackBeard Editor und was ist Markdown?

BlackBeard Editor ist ein Programm für den Mac, mit dem Sie Textdateien im Format Markdown bequem schreiben und lesen können.

Markdown ist kein eigenes Programm, sondern eine Art, Text zu markieren. Zum Beispiel:

  • # Überschrift — große Überschrift
  • **fett** — fetter Text
  • - Punkt — Listenpunkt

BlackBeard Editor zeigt entweder diese „Markierungen“ (Symbole) oder die fertige Ansicht — wie auf einer Webseite oder in einer PDF.

Erster Start — was Sie anklicken

Nach dem Öffnen von BlackBeard Editor sehen Sie einen Begrüßungsbildschirm mit drei Aktionen:

  1. New Document — leere Seite für neuen Text.
  2. Open File — eine vorhandene Datei auf dem Mac wählen (oft mit der Endung .md).
  3. Import — Datei aus dem Finder ins BlackBeard Editor-Fenster ziehen.

Zuletzt geöffnet: Unter den Schaltflächen kann eine Liste erscheinen mit dem, was Sie früher geöffnet haben. Klicken Sie auf den Namen — die Datei öffnet sich wieder. Mit Clear wird nur die Liste geleert; die Dateien auf der Festplatte bleiben erhalten.

Text schreiben und formatieren

Oben im Fenster gibt es eine Symbolleiste mit Schaltflächen. Sie fügen die Formatierung für Sie ein:

  • fett und kursiv;
  • Überschriften und Listen;
  • Link, Bild, Codeblock.

Sie können die Symbole auch selbst eingeben — zum Beispiel **Wort** für fett.

Einen Fehler können Sie rückgängig machen: Menü „Bearbeiten“ → „Rückgängig“ oder die Mac-Tastenkombination (siehe Abschnitt unten).

Vier Modi — was sie bedeuten

Oben können Sie den Modus wechseln. Es ist derselbe Text — nur die Darstellung ändert sich.

Edit

Normaler Text mit Symbolen (#, ** usw.). Praktisch, wenn viel Code dabei ist oder Sie die Formatierung von Hand anpassen.

Live

Der Text sieht sofort „schön“ aus — Überschriften groß, Fettdruck wirklich fett. Gut für Artikel und Notizen.

Split

Links der Text mit Symbolen, rechts die Vorschau. Gut zum Lernen und für lange Dokumente.

Preview

Nur die fertige Ansicht, ohne Symbole. Praktisch vor dem Drucken oder Export.

Tipp: Bei langen Dokumenten den Modus Split nutzen und unter Settings → Appearance das synchrone Scrollen aktivieren, damit linke und rechte Seite gemeinsam scrollen.

Speichern

BlackBeard Editor speichert Ihre Arbeit als normale Textdatei auf dem Mac.

  1. Menü File → Save (oder die übliche Mac-Tastenkombination — siehe unten).
  2. Unter Settings → General können Sie die automatische Speicherung einschalten — dann speichert das Programm alle 10–120 Sekunden von selbst.

Wenn Sie Bilder einfügen, kann die App sie in den Ordner des Dokuments kopieren — dann öffnet sich die Datei auch auf einem anderen Mac richtig (Einstellung unter Settings → Editor).

Überschriftenliste (Outline)

Wenn Ihr Text Überschriften hat (#, ##, …), öffnen Sie in der Leiste die Schaltfläche für Outline.

Es erscheint eine Liste der Abschnitte. Klicken Sie auf einen Namen — das Dokument scrollt dorthin. Am Rand der Leiste können Sie ziehen, um sie breiter oder schmaler zu machen.

Mehrere Dokumente gleichzeitig

Sie können mehrere Dateien in Registerkarten öffnen (wie im Browser). Jede Registerkarte hat eigenen Text. Beim Wechseln der Registerkarten bleibt die Rückgängig-Historie pro Datei erhalten.

Farben und Design

Möchten Sie einen dunklen oder hellen Bildschirm? Öffnen Sie die Liste Themes im Menü oder in der Leiste. Wählen Sie, was angenehm für die Augen ist.

Auf dieser Webseite schaltet die Schaltfläche ◐ oben nur das helle/dunkle Design der Seite um — nicht in BlackBeard Editor selbst.

Settings — was wo liegt

Öffnen Sie Settings im BlackBeard Editor-Menü. Die wichtigsten Bereiche:

  • General — automatisches Speichern: ein/aus und wie oft.
  • Editor — Schriftgröße, Zeilennummern, Tabulatoren oder Leerzeichen.
  • Typing — Rechtschreibprüfung; „intelligente Anführungszeichen“ (für technische Texte oft besser aus).
  • Appearance — im Modus Split Text und Vorschau gemeinsam scrollen.
  • Export — Einstellungen für PDF und Bilder.

Tastenkürzel (am Mac)

Die Taste ⌘ heißt Command (Symbol ⌘, neben der Leertaste). In BlackBeard Editor gibt es vertraute Kombinationen:

  • ⌘N — New Document
  • ⌘O — Open File
  • ⌘S — speichern
  • ⌘Z — rückgängig · ⌘⇧Z — wiederholen
  • ⌘B fett · ⌘I kursiv
  • ⌘K — vollständige Liste der Tastenkürzel in der App

Sie müssen sie nicht auswendig lernen — alles geht auch mit der Maus und über die Menüs.

Export als PDF und andere Formate

Wenn der Text fertig ist:

  1. Menü File öffnen (oder „Teilen“).
  2. Format wählen: PDF, Webseite (HTML), Bild (PNG) oder Word (DOCX).

Für eine E-Book-Datei (EPUB) brauchen Sie das kostenlose Programm Pandoc auf dem Mac.

Formeln und Diagramme (falls nötig)

Das ist optional. Für eine Formel im Text nutzt man Dollarzeichen, zum Beispiel $E = mc^2$. Für ein einfaches Flussdiagramm — einen Codeblock mit dem Wort mermaid. Die App zeigt sie in Preview ohne Internet.

Wenn Sie keine Formeln und Diagramme brauchen — diesen Abschnitt können Sie überspringen. Das Schiff segelt auch so.

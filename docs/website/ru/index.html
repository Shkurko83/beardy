Пишите журналы, как капитан

BlackBeard Editor — редактор Markdown для Mac: обычный текст превращается в аккуратные документы. Заметки, README, корабельные уставы… попугая покормите потом — пишите сейчас.

Захватить приложение (бесплатно) Как плавать в редакторе

Файлы остаются на вашем Mac. Автосохранение не даст сокровищам утонуть. Просмотр работает без интернета — берег не обязателен.

Указ капитана

Это приложение неподъёмно дорого для честного купца — потому что цена ноль дублонов. Купить то, что никто не продавал, нельзя. Остаётся единственно разумное:

Спиратить его здесь. Законно. Для Mac.

  • Ни восьмых реала. Ни подписки. Ни королевской пошлины.
  • Мы не призрак Чёрной Бороды — но на лого золотой зуб. Считайте предупреждением.
  • По доске сойдёте только если закроете приложение без сохранения.
  • Украдьте редактор у нас. Мы сами просим.

План абордажа в три шага

И сухопутный матрос поднимет парус за пару минут.

  1. Захватите корабль. Установите BlackBeard Editor на Mac и откройте — новый журнал, файл с диска или перетащите его в окно.
  2. Чертите карты. Пишите кнопками на панели или знаками Markdown — заголовки, списки, код для команды.
  3. Выберите подзорную трубу. Только текст, только красивый вид или оба варианта рядом — потом PDF, когда рейд окончен.

Полный морской устав

Кто плывёт с нами?

Разработчики, писцы и все, кто на Mac пишет длинные тексты.

Кодовые корсары

  • Цветная подсветка кода в свитках
  • Прыжок по разделам — список заголовков сбоку
  • Отмена действий переживает смену режима просмотра
  • Много вкладок; автосохранение, как у бертольда

Писцы и летописцы

  • Режим Live — оформление видно сразу при наборе
  • Экспорт в PDF, веб-страницу, картинку или Word
  • Формулы и схемы в трюме — без сети
  • Орфография, чтобы не только «арр» писалось верно

Что в трюме

Добыча — без флотского канцелярита.

Четыре режима просмотра

Edit, Live, Split, Preview — один документ, разные трубы. Переключайте свободно; история отмены с вами.

Сабли на панели

Жирный, заголовки, списки, ссылки, код — по одному клику, если руны не заучили.

Карта заголовков

Сбоку список разделов. Клик — и свиток прокручивается туда. Карта сокровищ не нужна.

Математика и морские карты

Формулы и диаграммы Mermaid в просмотре — работает, когда вы в открытом море без сети.

Тёмная и золотая тема

Светлая, тёмная или как в системе Mac — ночной рейд или полуденная попытка.

Экспорт свитков

PDF, HTML, PNG, DOCX — и EPUB, если поставить Pandoc (бесплатно, как этот редактор).

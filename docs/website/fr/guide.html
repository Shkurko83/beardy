Comment utiliser Beardy

Ce guide est rédigé en langage simple. Si vous n’avez jamais utilisé Markdown — ce n’est pas grave. Suivez les étapes dans l’ordre.

Qu’est-ce que Beardy et qu’est-ce que Markdown ?

Beardy est une application pour Mac qui permet d’écrire et de lire facilement des fichiers texte au format Markdown.

Markdown n’est pas un programme à part : c’est une façon de marquer le texte. Par exemple :

  • # Titre — grand titre
  • **gras** — texte en gras
  • - élément — point de liste

Beardy affiche soit ces « marques » (symboles), soit le rendu final — comme sur un site web ou dans un PDF.

Premier lancement — quoi cliquer

Après avoir ouvert Beardy, vous voyez un écran d’accueil avec trois actions :

  1. Nouveau document — page vide pour un nouveau texte.
  2. Ouvrir un fichier — choisir un fichier déjà présent sur le Mac (souvent avec l’extension .md).
  3. Importer — glisser un fichier depuis le Finder dans la fenêtre Beardy.

Fichiers récents : sous les boutons, une liste peut afficher ce que vous avez ouvert auparavant. Cliquez sur le nom — le fichier se rouvre. Le bouton Effacer vide seulement la liste ; les fichiers sur le disque ne sont pas supprimés.

Écrire et mettre en forme le texte

En haut de la fenêtre, une barre d’outils avec des boutons insère la mise en forme pour vous :

  • gras et italique ;
  • titres et listes ;
  • lien, image, bloc de code.

Vous pouvez aussi taper les symboles vous-même — par exemple **mot** pour le gras.

Une erreur peut être annulée : menu « Édition » → « Annuler » ou le raccourci Mac (voir la section ci-dessous).

Quatre modes — ce qu’ils signifient

En haut, vous pouvez changer de mode. C’est le même texte — seule la façon de le voir change.

Edit

Texte normal avec les symboles (#, **, etc.). Pratique quand il y a beaucoup de code ou que vous modifiez la mise en forme à la main.

Live

Le texte a tout de suite un bel aspect — titres grands, gras vraiment gras. Pratique pour les articles et les notes.

Split

À gauche le texte avec symboles, à droite l’aperçu. Idéal pour apprendre et pour les longs documents.

Preview

Uniquement le rendu final, sans symboles. Pratique avant l’impression ou l’export.

Conseil : pour un long document, activez le mode Split et dans « Réglages → Apparence » — « défilement synchronisé », pour que la partie gauche et la droite défilent ensemble.

Comment enregistrer

Beardy enregistre votre travail comme un fichier texte ordinaire sur votre Mac.

  1. Menu Fichier → Enregistrer (ou le raccourci Mac habituel — voir ci-dessous).
  2. Dans Réglages → Général, vous pouvez activer l’enregistrement automatique — le programme enregistre alors toutes les 10 à 120 secondes.

Si vous insérez des images, l’application peut les copier dans le dossier du document — le fichier s’ouvrira correctement sur un autre Mac aussi (réglage dans « Réglages → Éditeur »).

Liste des titres (Outline)

Si votre texte contient des titres (#, ##, …), ouvrez dans la barre le bouton structure / titres.

Une liste des sections apparaît. Cliquez sur un nom — le document défile jusqu’à cet endroit. Vous pouvez tirer le bord du panneau pour l’élargir ou le rétrécir.

Plusieurs documents en même temps

Vous pouvez ouvrir plusieurs fichiers dans des onglets (comme dans un navigateur). Chaque onglet a son propre texte. Changer d’onglet ne efface pas l’historique « annuler » de chaque fichier.

Couleurs et thème

Vous voulez un écran sombre ou clair ? Ouvrez la liste des thèmes dans le menu ou la barre (dans l’app, cela peut s’appeler « Themes »). Choisissez ce qui repose les yeux.

Sur ce site, le bouton ◐ en haut change seulement le thème clair/sombre du site — pas celui de Beardy.

Réglages — où trouver quoi

Ouvrez Réglages dans le menu Beardy. Les sections principales :

  • Général — enregistrement automatique : activé/désactivé et fréquence.
  • Éditeur — taille de police, numéros de ligne, tabulations ou espaces.
  • Saisie — correction orthographique ; « guillemets intelligents » (souvent mieux désactivés pour les textes techniques).
  • Apparence — en mode Split, faire défiler le texte et l’aperçu ensemble.
  • Export — paramètres PDF et images.

Raccourcis clavier (sur Mac)

La touche ⌘ est Commande (symbole ⌘, à côté de la barre d’espace). Dans Beardy, les combinaisons habituelles :

  • ⌘N — nouveau document
  • ⌘O — ouvrir un fichier
  • ⌘S — enregistrer
  • ⌘Z — annuler · ⌘⇧Z — rétablir
  • ⌘B gras · ⌘I italique
  • ⌘K — liste complète des raccourcis dans l’application

Inutile de tout mémoriser — tout se fait aussi à la souris et via les menus.

Export en PDF et autres formats

Quand le texte est prêt :

  1. Ouvrez le menu Fichier (ou « Partager »).
  2. Choisissez le format : PDF, page web (HTML), image (PNG) ou Word (DOCX).

Pour un fichier livre (EPUB), il faut le programme gratuit Pandoc sur Mac.

Formules et schémas (si besoin)

C’est facultatif. Pour une formule dans le texte, on utilise des signes dollar, par exemple $E = mc^2$. Pour un schéma simple — un bloc de code avec le mot mermaid. L’application les affiche dans l’aperçu sans Internet.

Si vous n’avez pas besoin de formules ni de schémas — vous pouvez ignorer cette section.

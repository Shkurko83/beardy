Comment utiliser BlackBeard Editor

Ce guide est écrit en langage simple. Si vous n’avez jamais utilisé Markdown — pas de panique. Suivez les étapes dans l’ordre, capitaine.

Qu’est-ce que BlackBeard Editor et qu’est-ce que Markdown ?

BlackBeard Editor est un programme pour Mac qui aide à écrire et lire des fichiers texte au format Markdown.

Markdown n’est pas un programme à part — c’est une façon de marquer le texte. Par exemple :

  • # Titre — grand titre
  • **gras** — texte en gras
  • - élément — puce de liste

BlackBeard Editor affiche soit les « marques » (symboles), soit le rendu final — comme sur le web ou en PDF.

Premier lancement — quoi cliquer

À l’ouverture de BlackBeard Editor, vous voyez l’écran d’accueil avec trois actions :

  1. New Document — page vide pour un nouveau texte.
  2. Open File — ouvrir un fichier existant sur votre Mac (souvent avec l’extension .md).
  3. Import — glisser un fichier depuis le Finder dans la fenêtre BlackBeard Editor.

Fichiers récents : sous les boutons, une liste de ce que vous avez déjà ouvert peut apparaître. Cliquez sur un nom pour le rouvrir. Le bouton Clear efface seulement la liste ; les fichiers sur le disque ne sont pas supprimés.

Comment écrire et mettre en forme

En haut de la fenêtre, une barre d’outils avec des icônes insère la mise en forme pour vous :

  • gras et italique ;
  • titres et listes ;
  • lien, image, bloc de code.

Vous pouvez aussi taper les symboles vous-même — par exemple **mot** pour le gras.

Pour annuler une erreur : menu Édition → Annuler, ou le raccourci Mac habituel (voir ci-dessous).

Quatre modes — ce qu’ils signifient

En haut, vous pouvez changer de mode. C’est le même document — seul l’affichage change.

Edit

Texte brut avec symboles (#, **, etc.). Idéal quand il y a beaucoup de code ou que vous retouchez la syntaxe à la main.

Live

Le texte a l’air « fini » pendant que vous tapez — titres grands, gras vraiment gras. Pratique pour articles et notes.

Split

À gauche le texte avec symboles, à droite l’aperçu. Bien pour apprendre Markdown et pour les longs documents.

Preview

Seulement le rendu final, sans symboles. Pratique avant impression ou export.

Astuce : pour un long document, utilisez Split et activez le défilement synchronisé dans Settings → Appearance — les deux côtés défilent ensemble.

Comment enregistrer

BlackBeard Editor enregistre votre travail comme fichier texte normal sur votre Mac.

  1. Menu File → Save (ou le raccourci Mac standard — voir ci-dessous).
  2. Dans Settings → General, vous pouvez activer l’enregistrement automatique — l’app sauvegarde alors toutes les 10 à 120 secondes.

Si vous insérez des images, l’app peut les copier dans le dossier du document — le fichier s’ouvrira correctement sur un autre Mac aussi (réglage dans Settings → Editor).

Liste des titres (Outline)

Si votre document a des titres (#, ##, …), ouvrez le panneau Outline dans la barre d’outils.

Vous verrez la liste des sections. Cliquez sur un nom — le document défile jusqu’à cet endroit. Vous pouvez tirer le bord du panneau pour l’élargir ou le rétrécir.

Plusieurs documents à la fois

Vous pouvez ouvrir plusieurs fichiers dans des onglets (comme dans un navigateur). Chaque onglet est un document distinct. Changer d’onglet n’efface pas l’historique d’annulation de chaque fichier.

Couleurs et thème

Écran sombre ou clair ? Ouvrez la liste Themes dans le menu ou la barre d’outils. Choisissez ce qui repose les yeux.

Sur ce site, le bouton ◐ en haut ne change que le thème du site — pas celui de BlackBeard Editor.

Settings — où trouver quoi

Ouvrez Settings depuis le menu BlackBeard Editor. Onglets principaux :

  • General — enregistrement automatique : activé/désactivé et fréquence.
  • Editor — taille de police, numéros de ligne, tabulations ou espaces.
  • Typing — correction orthographique ; « guillemets intelligents » (souvent à désactiver pour le texte technique).
  • Appearance — faire défiler texte et aperçu ensemble en mode Split.
  • Export — options pour PDF et images.

Raccourcis clavier (sur Mac)

La touche ⌘ s’appelle Command (symbole ⌘, à côté de la barre d’espace). BlackBeard Editor propose des combinaisons familières :

  • ⌘N — New Document
  • ⌘O — Open File
  • ⌘S — enregistrer
  • ⌘Z — annuler · ⌘⇧Z — rétablir
  • ⌘B gras · ⌘I italique
  • ⌘K — liste complète des raccourcis dans l’app

Inutile de tout mémoriser — tout se fait aussi à la souris et via les menus.

Exporter en PDF et autres formats

Quand le texte est prêt :

  1. Ouvrez le menu File (ou Partager).
  2. Choisissez le format : PDF, page web (HTML), image (PNG) ou Word (DOCX).

Pour un fichier livre (EPUB), installez le programme gratuit Pandoc sur votre Mac.

Formules et schémas (si besoin)

C’est optionnel. Pour une formule dans le texte, utilisez des signes dollar, par exemple $E = mc^2$. Pour un organigramme simple — un bloc de code avec le mot mermaid. L’app les affiche en Preview sans internet.

Pas besoin de maths ni de schémas ? Passez cette section — le navire avance quand même.

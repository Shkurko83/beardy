Ohé, Mac · Cabine de Barbe Noire

Écrivez vos journaux de bord comme un capitaine

BlackBeard Editor est un éditeur Markdown pour Mac : du texte brut qui devient de beaux documents. Notes, README, règlements de bord… nourrissez le perroquet plus tard ; écrivez maintenant.

Flibustier l’app (gratuit) Apprendre à gouverner l’éditeur

Vos fichiers restent sur votre Mac. L’autosauvegarde évite que le butin coule à pic. L’aperçu fonctionne hors ligne — pas besoin de voir la côte.

Ordre du capitaine

Cette app est ruineusement chère pour un marchand honnête — parce que le prix est de zéro doublon. On n’achète pas ce qu’on ne vend pas. Il ne reste donc qu’une solution raisonnable :

Flibustier-la ici. Légalement. Sur Mac.

  • Pas un réal de huit. Pas d’abonnement. Pas de taxe royale.
  • Nous ne sommes pas le fantôme de Barbe Noire — mais le logo a une dent en or. Considérez-vous prévenu.
  • La planche, c’est seulement si vous fermez sans enregistrer.
  • Volez-nous l’éditeur. Nous l’exigeons.

Plan d’abordage en trois temps

Même un terrien hisse les voiles en quelques minutes.

  1. Prenez le navire. Installez BlackBeard Editor sur votre Mac et ouvrez-le — nouveau journal, fichier existant ou glisser-déposer dans la fenêtre.
  2. Tracez les cartes. Écrivez avec la barre d’outils ou les signes Markdown — titres, listes, code pour l’équipage.
  3. Choisissez la longue-vue. Texte seul, bel aperçu seul, ou les deux côte à côte — puis export PDF quand l’expédition est finie.

Ce qu’il y a en cale

Le butin — sans jargon de la marine royale.

Quatre modes d’affichage

Edit, Live, Split, Preview — un document, plusieurs longues-vues. Basculez à volonté ; l’historique d’annulation vous suit.

Sabres de la barre d’outils

Gras, titres, listes, liens, code — un clic si vous n’avez pas mémorisé les runes.

Carte des titres

Panneau latéral des sections. Un clic — le parchemin défile jusqu’au bon endroit. Pas besoin de carte au trésor.

Maths & cartes marines

Formules et diagrammes Mermaid dans l’aperçu — même en pleine mer sans réseau.

Sombre & or

Clair, sombre ou comme votre Mac — raid de nuit ou négociation de midi.

Exporter les parchemins

PDF, HTML, PNG, DOCX — et EPUB si vous installez Pandoc (gratuit, comme cet éditeur).

Ahoy, Mac · Blackbeard's writing deck

Write your logbooks like a captain

BlackBeard Editor is a Markdown editor for Mac — plain text that turns into handsome documents. Notes, README files, ship's articles… feed the parrot later; write now.

Boarding plan in three steps

Even a landlubber can hoist sail in minutes.

  1. Take the ship. Install BlackBeard Editor on your Mac and open it — new log, open a file, or drag one aboard.
  2. Scratch your charts. Write with toolbar buttons or Markdown marks — headings, lists, code for the crew.
  3. Choose your spyglass. Text only, pretty view, or split deck — then export to PDF when the voyage is done.

The loot — explained without navy jargon.

Four view modes

Edit, Live, Split, Preview — same document, different spyglasses. Switch freely; undo stays with you.

Toolbar cutlasses

Bold, headings, lists, links, code — one click each if you do not memorize the runes.

Chart of headings

A side panel lists your sections. Click — the scroll jumps there. No treasure map required.

Math & sea charts

Formulas and Mermaid diagrams render in preview — still works when you're off the grid.

Dark & gold themes

Export scrolls

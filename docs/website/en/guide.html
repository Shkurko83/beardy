How to use Beardy

This guide is written in simple language. If you have never used Markdown — that is fine. Follow the steps in order.

What is Beardy and what is Markdown?

Beardy is a program for Mac that helps you write and read text files in Markdown format.

Markdown is not a separate program — it is a way to mark up text. For example:

  • # Title — large heading
  • **bold** — bold text
  • - item — bullet list

Beardy shows you either the “marks” (symbols) or the finished look — like in a browser or PDF.

First launch — what to press

When you open Beardy, you see the welcome screen with three main actions:

  1. New Document — start with an empty page.
  2. Open File — open an existing file from your Mac (often with the ending .md).
  3. Import — drag a file from Finder into the Beardy window.

Recent files: below the buttons you may see files you opened before. Click a name to open it again. The Clear button removes the list (files on disk are not deleted).

How to write and format text

At the top of the window there is a toolbar with icons. They insert formatting for you:

  • bold and italic;
  • headings and lists;
  • link, image, block of code.

You can also type symbols yourself — for example **word** for bold.

To undo a mistake: menu Edit → Undo, or the usual Mac shortcut (see section below).

Four view modes — what each one means

At the top you can switch the mode. It is the same document — only the display changes.

Edit

You see plain text with symbols (#, **, and so on). Best when you copy code or edit symbols by hand.

Live

Text looks “finished” while you type — headings are large, bold is bold. Convenient for articles and notes.

Split

Left — text with symbols, right — how it will look. Good for learning Markdown and for long documents.

Preview

Only the finished look, without symbols. Convenient before printing or export.

Tip: for a long document use Split and turn on “sync scroll” in Settings → Appearance — both sides scroll together.

How to save

Beardy saves your work as a normal text file on your Mac.

  1. Menu File → Save (or the standard Mac save shortcut — see below).
  2. In Settings → General you can turn on auto-save — then the app saves every 10–120 seconds by itself.

If you insert images, the app can copy them into the document folder — then the file will open correctly on another Mac too (setting in Settings → Editor).

Headings list (Outline)

If your document has headings (#, ##, …), open the headings panel on the toolbar.

You will see a list of section names. Click a name — the document scrolls to that place. You can drag the edge of the panel to make it wider or narrower.

Several documents at once

You can open multiple files in tabs (like in a browser). Each tab is a separate document. Switching tabs does not erase your undo history in each file.

Colors and theme

Do you want a dark or light screen? Open the theme list from the menu or toolbar (in the app it may be called “Themes”). Pick a look you like.

On this website, the circle button at the top (◐) switches light / dark / system theme for the site only — not inside Beardy.

Settings — what is where

Open Settings from the Beardy menu or toolbar. Main tabs:

  • General — auto-save on/off and how often.
  • Editor — font size, line numbers, tabs or spaces.
  • Typing — spell check; “smart quotes” (often better to leave off for technical text).
  • Appearance — scroll text and preview together in Split mode.
  • Export — options for PDF and images.

Keyboard shortcuts (for Mac)

On Mac, the ⌘ key is called Command (⌘ symbol, next to the space bar). Beardy supports common shortcuts:

  • ⌘N — new document
  • ⌘O — open file
  • ⌘S — save
  • ⌘Z — undo · ⌘⇧Z — redo
  • ⌘B bold · ⌘I italic
  • ⌘K — full list of shortcuts inside the app

You do not have to memorize them — everything can be done with the mouse and menus.

Export to PDF and other formats

When the text is ready:

  1. Open menu File (or Share).
  2. Choose format: PDF, web page (HTML), picture (PNG), or Word (DOCX).

For a book file (EPUB) you need the free program Pandoc installed on your Mac.

Formulas and charts (if you need them)

This is optional. For a formula in text use dollar signs, for example $E = mc^2$. For a simple flowchart — a code block with the word mermaid. The app shows them in preview without internet.

If you do not need math or charts — you can skip this section.
